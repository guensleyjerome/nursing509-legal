⚠️ Supprimer mon compte Nursing 509

ATTENTION: Cette action est permanente et irréversible

En supprimant votre compte, vous perdrez définitivement:

  • Votre compte et toutes vos données personnelles
  • Tout votre historique de quiz et progression
  • Vos questions sauvegardées et flashcards
  • Votre accès PREMIUM (si vous l'avez acheté - aucun remboursement)
  • Tous vos résultats et statistiques

Il n'y a AUCUN moyen de restaurer votre compte ou vos achats après la suppression.

Comment supprimer mon compte

Dans l'application Nursing 509:

  1. Ouvrez l'application
  2. Connectez-vous à votre compte
  3. Allez dans Paramètres (icône ⚙️)
  4. Faites défiler jusqu'à la section "Gestion des Données"
  5. Cliquez sur "Supprimer mon compte" (texte rouge)
  6. Lisez l'avertissement complet
  7. Confirmez en cliquant sur "OUI, SUPPRIMER DÉFINITIVEMENT"
  8. Votre compte sera immédiatement supprimé

Que se passe-t-il après la suppression?

Données supprimées immédiatement:

  • ✅ Compte Firebase Authentication supprimé
  • ✅ Profil utilisateur supprimé de notre base de données
  • ✅ Historique de paiements supprimé
  • ✅ Progression et résultats supprimés
  • ✅ Toutes les données locales effacées de votre appareil

Données conservées:

  • Aucune - toutes vos données sont supprimées définitivement

Vous avez changé d'avis?

Si vous souhaitez juste:

  • Réinitialiser votre progression - Utilisez "Réinitialiser la progression" dans Paramètres (conserve votre compte)
  • Se déconnecter - Utilisez "Déconnexion" (conserve vos données sur nos serveurs)
