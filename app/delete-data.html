Supprimer mes données - Nursing 509

ℹ️ Note importante:

Cette fonction vous permet de supprimer votre progression et historique de quiz SANS supprimer votre compte. Votre compte reste actif, mais vos résultats sont effacés.

Réinitialiser la progression (sans supprimer le compte)

Dans l'application Nursing 509:

  1. Ouvrez l'application
  2. Connectez-vous à votre compte
  3. Allez dans Paramètres (icône ⚙️)
  4. Faites défiler jusqu'à la section "Gestion des Données"
  5. Cliquez sur "Réinitialiser la progression"
  6. Confirmez la réinitialisation

Qu'est-ce qui est supprimé?

✅ Données supprimées:

  • Historique de tous vos quiz
  • Résultats et scores
  • Questions sauvegardées
  • Progression des flashcards
  • Statistiques d'étude

✓ Données conservées:

  • Votre compte utilisateur
  • Vos informations personnelles (nom, téléphone, email)
  • Votre statut premium (si activé)
  • Votre code d'activation

Différence avec "Supprimer mon compte"

Réinitialiser progression Supprimer compte
Compte supprimé ❌ Non ✅ Oui
Progression effacée ✅ Oui ✅ Oui
Premium conservé ✅ Oui ❌ Non
Peut se reconnecter ✅ Oui ❌ Non
